Pada hasil run 1 terdapat pembelian berjumlah 2 pasang sepatu, sehingga mendapatkan diskon 50000. Total harga awal adalah 1200000. Karena mendapatkan diskon/potongan harga sebesar 50000, maka (harga awal - potongan harga) adalah 1200000-50000 = 1150000. Sehingga total akhir dan uang yang harus dibayarkan adalah 1150000.
